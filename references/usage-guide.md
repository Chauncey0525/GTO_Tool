# GTO Poker Analyzer 使用指南

## 概览
GTO Poker Analyzer 是一个专业的德州扑克策略分析工具，帮助玩家学习和应用博弈论最优（GTO）策略。使用专业位置命名系统，提供严格的游戏流程控制。

## 核心功能

### 1. 游戏初始化

**第一步：设置游戏参数**
- **游戏人数**：选择2-9人桌
- **你的位置**：从动态生成的位置列表中选择
- **盲注设置**：SB/BB/Ante

**位置系统说明**

游戏使用国际通用的专业位置命名：

| 缩写 | 全称 | 中文名 | 描述 |
|------|------|--------|------|
| UTG | Under the Gun | 枪口位 | 翻牌前第一个行动 |
| UTG+1 | UTG+1 | 枪口+1位 | 第二个行动（8-9人桌） |
| UTG+2 | UTG+2 | 枪口+2位 | 第三个行动（9人桌） |
| LJ | LoJack | 劫持位前一位 | 中前位置 |
| HJ | Hijack | 劫持位 | 中后位置 |
| CO | Cutoff | 关煞位/庄前位 | 庄家前一位置 |
| BTN | Button | 庄位 | 最佳位置 |
| SB | Small Blind | 小盲 | 翻牌后第一个行动 |
| BB | Big Blind | 大盲 | 翻牌前最后行动 |

**不同人数的位置配置**

- **2人桌**：BTN, BB
- **3人桌**：BTN, SB, BB
- **4人桌**：CO, BTN, SB, BB
- **5人桌**：HJ, CO, BTN, SB, BB
- **6人桌**：LJ, HJ, CO, BTN, SB, BB
- **7人桌**：UTG, LJ, HJ, CO, BTN, SB, BB
- **8人桌**：UTG, UTG+1, LJ, HJ, CO, BTN, SB, BB
- **9人桌**：UTG, UTG+1, UTG+2, LJ, HJ, CO, BTN, SB, BB

### 2. 玩家信息管理

- **位置显示**：每个位置显示缩写和中文名称
- **你的位置高亮**：你选择的位置会以绿色高亮显示
- **当前行动玩家**：当前行动的玩家会以红色高亮显示
- **筹码管理**：输入每个玩家的后手筹码（BB为单位）
- **状态切换**：在局/弃牌/全下

### 3. 牌局输入

- **手牌选择**：点击扑克牌网格选择2张手牌
- **公共牌选择**：根据街道选择0-5张公共牌
- **街道切换**：翻牌前 → 翻牌 → 转牌 → 河牌

### 4. 行动流程控制

**严格的行动顺序**

系统按照标准的德州扑克规则控制行动顺序：

**翻牌前（Preflop）**
- 从UTG开始，依次行动：UTG → UTG+1 → LJ → HJ → CO → BTN → SB → BB

**翻牌后（Postflop）**
- 从SB开始，依次行动：SB → BB → UTG → UTG+1 → ... → BTN

**行动类型**
- Fold：弃牌
- Check：过牌（无需跟注时）
- Call：跟注（需跟注时）
- Raise：加注

**操作步骤**
1. 系统自动提示当前行动玩家
2. 显示需要跟注的金额
3. 选择行动类型
4. 如选择Raise，输入加注金额
5. 系统自动移动到下一个玩家

### 5. 底池状态分析

**实时计算**
- 盲注投入：SB + BB
- 前注总额：Ante × 人数
- 已下注额：所有玩家的下注总和
- 当前底池：总和
- 需跟注：当前行动玩家需要跟注的金额

**底池赔率**
- 底池赔率百分比
- 最小胜率要求：盈利所需的最小equity
- SPR（Stack-to-Pot Ratio）：有效筹码底池比
- 底池大小（BB）

### 6. 下注建议

根据当前底池自动计算建议下注额：
- 33% Pot：小额下注
- 50% Pot：标准下注
- 75% Pot：大额下注
- 100% Pot：底池大小下注

### 7. GTO策略分析

**行动频率**
- Fold%：弃牌频率
- Call%：跟牌/过牌频率
- Raise%：加注/下注频率

**期望值（EV）**
- 每种行动的期望收益（BB为单位）

**策略洞察**
- 手牌强度分析
- 位置优势考虑
- 底池赔率解读
- 街道阶段策略

## 使用流程

### 标准分析流程

1. **打开HTML文件**
   - 在浏览器中打开gto-analyzer.html

2. **初始化游戏**
   - 选择游戏人数（2-9人）
   - 选择你的位置（会显示位置说明）
   - 设置盲注和前注
   - 点击"开始游戏"

3. **调整玩家信息**
   - 调整各位置筹码（可选）
   - 你的位置会以绿色高亮

4. **选择手牌**
   - 点击扑克牌网格选择2张手牌

5. **记录行动**
   - 系统会提示当前行动玩家
   - 按顺序选择行动（Fold/Check/Call/Raise）
   - 系统自动推进到下一个玩家

6. **切换街道**
   - 当前街道行动完成后，点击"进入下一阶段"
   - 选择公共牌
   - 继续记录行动

7. **查看策略**
   - 点击"查看GTO策略"
   - 查看策略建议和期望值

## 高级技巧

### 1. 位置优势

- **BTN（庄位）**：最大位置优势，可扩大开牌范围
- **CO（关煞位）**：良好位置，可适当放宽
- **HJ/LJ**：中等位置，需要较强牌力
- **UTG**：最差位置，需要最强牌力
- **SB/BB**：翻牌前位置不利，翻牌后SB首先行动

### 2. 筹码深度策略

- **SPR > 10**：深筹码策略（可进行更多诈唬和浮动）
- **SPR 5-10**：中筹码策略（注重价值下注）
- **SPR < 5**：浅筹码策略（倾向于全下或弃牌）

### 3. 底池赔率计算

公式：底池赔率 = 跟注金额 / (底池 + 跟注金额)

示例：
- 底池10BB，需跟注5BB
- 底池赔率 = 5 / (10+5) = 33.3%
- 需要至少33.3%的胜率才能盈利

### 4. 行动路线分析

- 注意行动顺序和位置
- 分析对手的范围
- 考虑后续街道的打法

## 常见场景示例

### 场景1：翻牌前3-Bet决策

1. 初始化：选择6人桌，选择你的位置为BTN
2. 选择手牌：A♠K♠
3. 按行动顺序记录：
   - UTG: Fold
   - HJ: Fold
   - CO: Raise 3BB
   - 轮到你（BTN）
4. 查看GTO策略：建议3-Bet或Call

### 场景2：翻牌后C-Bet决策

1. 继续上一场景
2. 记录翻牌前行动
3. 进入翻牌阶段
4. 选择公共牌：K♠7♦2♥
5. 记录翻牌后行动
6. 查看C-Bet建议

### 场景3：河牌决策

1. 完整的翻牌前到转牌行动
2. 添加河牌
3. 记录对手下注
4. 分析跟注还是弃牌

## 常见问题

**Q: 如何重置分析？**
A: 点击右上角的"重置"按钮回到初始化界面。

**Q: 位置名称是什么意思？**
A: 使用国际通用的专业位置命名，如UTG=枪口位，BTN=庄位等。工具会显示中英文对照。

**Q: 行动顺序是怎样的？**
A: 翻牌前从UTG开始依次行动，翻牌后从SB开始。系统会自动控制行动顺序。

**Q: 如何选择我的位置？**
A: 在初始化界面选择，系统会根据游戏人数动态生成可选位置列表。

**Q: 策略建议准确吗？**
A: 基于简化GTO模型，提供参考建议。实际需结合对手风格和环境调整。

**Q: 支持多人底池分析吗？**
A: 支持，可记录多个玩家的行动，系统会按顺序控制行动流程。

**Q: 如何理解SPR？**
A: SPR = 有效筹码 / 底池，用于判断筹码深度，影响策略选择。
