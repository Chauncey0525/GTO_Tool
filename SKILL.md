# GTO Poker Analyzer

## 任务目标
- 本 Skill 用于：德州扑克GTO（博弈论最优）策略分析与学习
- 能力包含：完整的牌局输入、实时底池计算、策略分析、起手牌矩阵可视化
- 触发条件：用户需要分析德州扑克牌局、学习GTO策略、复盘牌局或询问最优打法时

## 前置准备
- 无需额外依赖
- 工具为纯前端HTML应用，可直接在浏览器中运行

## 操作步骤

### 1. 提供分析工具
向用户提供 `assets/gto-analyzer.html` 文件，用户可：
- 在浏览器中直接打开使用
- 无需服务器或网络连接
- 所有计算在本地完成

**第一步：游戏初始化**
用户打开工具后会看到初始化界面，需要设置：
1. **游戏人数**：选择2-9人桌
2. **你的位置**：从动态生成的位置列表中选择（系统会根据人数显示对应位置）
3. **盲注设置**：SB/BB/Ante

位置命名说明：
- **UTG** (Under the Gun) - 枪口位：翻牌前第一个行动
- **UTG+1** - 枪口+1位：第二个行动（8-9人桌）
- **UTG+2** - 枪口+2位：第三个行动（9人桌）
- **LJ** (LoJack) - 劫持位前一位：中前位置
- **HJ** (Hijack) - 劫持位：中后位置
- **CO** (Cutoff) - 关煞位/庄前位：庄家前一位置
- **BTN** (Button) - 庄位：最佳位置
- **SB** (Small Blind) - 小盲：翻牌后第一个行动
- **BB** (Big Blind) - 大盲：翻牌前最后行动

### 2. 功能说明
告知用户该工具包含以下核心功能：

**游戏设置**
- 游戏人数：支持2-9人桌（位置会根据人数动态调整）
- 你的位置：在初始化界面选择，游戏中会高亮显示
- 盲注设置：SB/BB可自定义
- 前注（Ante）：可选

**位置系统**
- 专业命名：UTG, UTG+1, UTG+2, LJ, HJ, CO, BTN, SB, BB
- 中文名称：枪口位、劫持位、庄位等
- 行动顺序：翻牌前从UTG开始，翻牌后从SB开始
- 你的位置会以绿色高亮显示
- 详见 [references/usage-guide.md](references/usage-guide.md) 中的位置系统说明

**玩家信息**
- 每个位置的筹码深度
- 本轮下注金额
- 玩家状态（在局/弃牌/全下）

**牌局输入**
- 手牌选择（2张）
- 公共牌选择（翻牌/转牌/河牌）
- 街道切换

**行动路线**
- 完整的行动记录（位置+动作+金额）
- 支持所有行动类型（Fold/Check/Call/Bet/Raise/All-in）

**底池分析**
- 实时底池计算
- 底池赔率显示
- 最小胜率要求
- SPR（筹码底池比）
- 筹码深度策略建议

**策略建议**
- Fold/Call/Raise频率
- 期望值（EV）计算
- 13x13起手牌矩阵
- 智能策略洞察

### 3. 使用指导
引导用户按照以下步骤使用：

1. 打开HTML文件
2. 在初始化界面设置：
   - 选择游戏人数（2-9人）
   - 选择你的位置（会显示位置说明）
   - 设置盲注
3. 点击"开始游戏"进入主界面
4. 调整各位置玩家筹码（可选）
5. 选择手牌
6. 按行动顺序记录行动
7. 查看底池状态和赔率
8. 点击"查看GTO策略"获取建议

### 4. 辅助分析（可选）
如用户需要，智能体可以：
- 解释GTO概念和策略原理
- 分析特定场景的最优打法
- 解读策略矩阵的含义
- 提供筹码管理建议

## 使用示例

**示例1：翻牌前AA的打法分析**
用户询问："我在BTN位置拿到AA，前面有人加注，我该怎么打？"

执行方式：
1. 提供HTML工具文件
2. 在初始化界面：选择6人桌，位置选择BTN
3. 点击"开始游戏"
4. 选择手牌：A♠A♥
5. 记录行动：前面玩家的加注（从UTG开始依次行动）
6. 分析策略：建议3-Bet或4-Bet，根据对手风格调整

**示例2：翻牌后的决策**
用户询问："我在CO位置有AK，翻牌K-7-2彩虹面，对手下注半池，我该跟注还是加注？"

执行方式：
1. 打开工具，选择手牌A♣K♣
2. 设置公共牌：K♠7♦2♥
3. 记录行动历史
4. 查看底池赔率和策略建议
5. 智能体补充分析：考虑对手范围、位置优势等因素

**示例3：筹码深度策略**
用户询问："我只有20个大盲，在BTN位置拿到TT，前面有人加注3BB，我该怎么办？"

执行方式：
1. 在工具中设置筹码深度为20BB
2. 选择手牌T♦T♠
3. 记录行动
4. 查看SPR分析（浅筹码策略）
5. 考虑全下或弃牌的二选一决策
